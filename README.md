# AE_SpiritBall "AE元気玉"

指定したaepファイルをaerender.exeに渡してレンダリングさせます。<br>


# Usage
有効なレンダーキューがあるaepファイルを、このアプリアイコンにドラッグします(リンクファイルでも可)<br>
インストールされているAEの一覧が表示されるので、バージョンを選んでクリック。
実行するかダイアログが表示されるのでOKで実行です。


アプリを立ち上げて、バージョンのアイコンにaepファイルをドラッグでもできます。
